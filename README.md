# 🧼 DataCleanPro

> “Clean data shouldn’t come with a dirty price tag.”

**DataCleanPro** is a friendly, affordable CSV cleaning tool built with Streamlit. It helps you clean messy spreadsheets without subscriptions or code.

---

## 🚀 Features

- Strip whitespace from string fields
- Normalize currency and date formats
- Remove duplicates and empty rows
- Fill missing values (Pro feature)
- View type distribution and logs
- Download cleaned file and cleaning report

---

## 💸 Pricing

| Rows Cleaned | Cost          |
|--------------|---------------|
| First 100    | Free          |
| 101–5,000    | $0.01 / 1,000 |
| 5,001+       | Tiered rates  |

No subscriptions. Pay only for what you clean.
